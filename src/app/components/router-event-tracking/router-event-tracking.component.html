Router Event Takibi

Component üzerinde hangi işlemin olduğu ya da başka componente geçiş durumlarının takibi için kullanabiliriz. Componentler arasında detaylı olarakta takip yapabiliriz.

Örnek için components/router-event dizinini inceleyebilirsiniz.

Angular Router Event Listesi

  • NavigationStart
  • RouteConfigLoadStart
  • RouteConfigLoadEnd
  • RoutesRecognized
  • GuardsCheckStart
  • ChildActivationStart
  • ActivationStart
  • GuardsCheckEnd
  • ResolveStart
  • ResolveEnd
  • ActivationEnd
  • ChildActivationEnd
  • NavigationEnd
  • Scroll
  • NavigationCancel
  • NavigationError
