Model Odaklı SetValue Fonksiyon Kullanımı

Modelimize veri atamak için kullanılır. Forma farklı bir değer atamak istiyorsak setValue fonksyonunu kullanırız.

Örnek için components/model-oriented-setvalue-function dizinini inceleyebilirsiniz.

this.userForm.controls["name"].setValue("Yasin Sunmaz");
