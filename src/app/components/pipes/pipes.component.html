Pipe Kullanımı

Pipe lar ile verilerimizi dönüştürebiliriz.

Örnek için components/pipes dizinini inceleyebilirsiniz.

Pipe oluşturmak için gereken kod satırı aşağıdaki gibidir. "pipe-name" kısmına istediğimiz ismi verebiliriz. 

 npm g p pipe-name

Uygulamalar

Uppercase | Büyük Harfe Çevirme

UpperCasePipe: Verilen string değerin tamamının büyük harfle gösterilmesini sağlar.

Örnek: {{ 'Harfleri büyüt' | uppercase}}

{{ lowerCaseValue | uppercase}}

LowerCasePipe | Küçük Harfe Çevirme

LowerCasePipe: Verilen string değerin tamamının küçük harfle gösterilmesini sağlar.

Örnek: {{ 'Harfleri KÜÇÜLT' | lowercase}}

{{ upperCaseValue | lowercase}}

TitleCasePipe | Başlık Formatına Çevirme

TitleCasePipe: Verilen string değerin başlık formatı şeklinde gösterilmesini sağlar.

Örnek: {{'angular pipe uygulaması | yasin sunmaz'| titlecase}}

{{ titleCaseValue | titlecase}}

SlicePipe | Metin Bölme

SlicePipe: Verilen dizi veya string değerin, gönderilen parametrelere göre bölünmesini sağlar.

Örnek: {{'yasinsunmaz'| slice:2:6}}

{{sliceCaseValue | slice:2:6}}

CurrencyPipe | Para Birimi

CurrencyPipe: Verilen sayı değerinin para birimi olarak gösterilmesini sağlar.

{{7.569 | currency}}

{{7.569 | currency:'CAD'}}

PercentPipe | Yüzde Biçimi Gösterme

PercentPipe: Verilen sayı değerinin yüzde biçiminde gösterilmesini sağlar.

{{0.259| percent}}

DecimalPipe | Ondalıklı Gösterme

DecimalPipe: Verilen sayı değerinin ondalıklı gösterilmesini sağlar.

{{6.256341| number}}

KeyValuePipe | ngFor Döngüsü

KeyValuePipe: Object veya Maplerin ngFor ile dönülmesini sağlar.

Örnek 1

{{country.value}}

Örnek 2
  • {{item.key}} ---> {{item.value}}

Keyvalue için sıralama yaparak verilerin daha düzgün görülmesini sağlayabiliriz.

  • {{item.key}} ---> {{item.value}}
DatePipe | Tarih Gösterme

DatePipe: Verilen tarih değerin, gönderilen parametrelere göre gösterilmesini sağlar.

{{1610301383709 | date:'hh:mm'}}

DatePipe: Verilen tarih değerin, gönderilen parametrelere göre gösterilmesini sağlar. Diğer kullanım şekilleri.

Saat: {{ nowDate | date:'hh:mm'}}
Gün, Ay ve Yıl için: {{ nowDate| date:'d.M.y'}}
Sözel Gün için(Örneğin Çarşamba): {{ nowDate| date:'EEEE'}}
Yılın kaçıncı haftası olduğunu öğrenmek için: {{ nowDate| date:'w'}}

JsonPipe

Nesnemizi json tipinde ekranda gösterebiliriz.

Orijinal Ekran Çıktısı:

{{ myObject }}

Pipe'lı Ekran Çıktısı

{{ myObject | json}}
