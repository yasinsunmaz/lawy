Component Kullanımı

Componentler bizim sayfalarımızı oluşturmak ve işlemleri yapabilmek için kullandığımız HTML çıktısı ve class diyebiliriz.

Örnek için components/component dizinini inceleyebilirsiniz.

ng generate component component-name

Kısa Kullanımı

ng g c component-name

Component Çağırma

HTML etiketi şeklinde açma ve kapatma ifadeleri ile çağırılır.

app-component
