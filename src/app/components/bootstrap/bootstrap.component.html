Bootstrap Kullanımı

Projenizde bootstrap kullanarak duyarlı, hoş ve güzel tasarımlar yapabilirsiniz. Boostrap kullanmak için paketi indirmeniz sonrasında dosyaları tanımlamanız gerekiyor.

Örnek için components/bootstrap dizinini inceleyebilirsiniz.

Paketin güncel versiyonunu indirmek için:

 {{"npm install bootstrap@next"}}

BootstrapMd paketi indirmek için:

 {{"npm install @angular/cdk --save"}} 
 {{"npm install angular-bootstrap-md --save"}}

angular.json dosyasında styles kısmınıda tanımlama yapmanız gerekiyor.

 {{"node_modules/angular-bootstrap-md/assets/scss/bootstrap/bootstrap.scss"}} 
 {{"node_modules/angular-bootstrap-md/assets/scss/mdb.scss"}} 
 {{"node_modules/bootstrap/dist/css/bootstrap.min.css"}}

Bootstrap ve BootrapMd versiyonu için package.json dosyasında dependencies altında bootstrap ve angular-bootstrap-md kısmına göz atababilirsiniz.

 {{"dependencies/bootstrap"}} 
 {{"dependencies/angular-bootstrap-md"}}

app.module.ts içerisinde tanımlama

 {{"imports: [MDBBootstrapModule.forRoot()]"}}

BootstrapMd içerisinde istediğiniz paketleri app.module.ts'de tanımlayarak projenizde kullanabilirsiniz. Örneğin buton, input, collapse, modal, dropdown için: {{"import { ButtonsModule, InputsModule,"+ +"CollapseModule, ModalModule, DropdownModule } from 'angular-bootstrap-md';"}}

 {{"import { ButtonsModule, InputsModule,"+
                    +"CollapseModule, ModalModule, DropdownModule } from 'angular-bootstrap-md';"}}

Import edilen paketleri çağırmamız yeterlidir.

 {{"imports: [MatButtonModule, InputsModule, CollapseModule, ,ModalModule, DropdownModule.forRoot()]"}}
